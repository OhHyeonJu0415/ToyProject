# ToyProject_Animation

HTML/CSS/Vanilla Javascript를 이용한 Rainbow Animation


📌 addEventListener

📌 !important

📌 cubic-bezier

📌 drop-shadow





---
#### :pencil2: 요구사항

1. 애니메이션 자동재생
   - 배경 서서히 바뀌기
   - 구름이 양 옆으로 사라지기
   - 무지개 펼쳐지기
   
2. 리플레이
   - 버튼 오버 시 색 바꾸기
   - 구름, 무지개가 처음부터 재생
