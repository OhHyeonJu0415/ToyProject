#### :pencil2: 요구사항

1. 개발자의 MBTI인 ENFP를 표현하기
    - 색감 많이 쓰기
    - 좋아하는 캐릭터를 CSS로 직접 그리기
    - 캐릭터에 애니메이션 효과 넣어서 생동감 표현하기
    - 돌아가는 꽃을 배경으로 넣기
    - 꽃 색에 맞춰서 배경 색 바꾸기
   
2. 글자에 애니메이션 효과 넣기
    - E,N,P가 살짝식 흔들리기
    - F가 360도 회전해서 재미 넣기

3. 글자에 마우스 오버 했을 때
    - 그림자가 위로 올라오면서 강조하기
    - 알파벳의 뜻을 아래에 표시하기
