# ToyProject_Animation

HTML/CSS/Vanilla Javascript를 이용한 Event Animation



📌 addEventListener

📌 background-position

📌 getComputedStyle



#### 요구사항

1. 사진 위에 마우스를 올렸을 때 사진이 옆으로 움직인다.
   
2. 사진을 눌렀을 때
   - 해당 계절의 상세보기 사진이 나온다.
   - 'click' 글자가 없어진다.
   - 모든 사진은 오른쪽 세로로 정렬된다.
   - 해당 계절의 미리보기 사진에 테두리가 생긴다.
   - 닫기 버튼이 생긴다.
   
3. 닫기 버튼을 눌렀을 때 처음화면으로 돌아간다.
